## 📦 데이터 흐름 핵심

### 학교 → 교육청 발송 (4가지 경로)
```
A. 이메일 (Gmail/Naver/Daum/Outlook)
B. 카톡 (PC/모바일)
C. 클라우드 드라이브 (Google Drive/OneDrive/NAS)
D. 사내 행정 시스템 (K-에듀파인 등)

→ 모든 경로 공통: 외부 채널 + 교육청 담당자가 SafeLoop 수신함에 수동 업로드
```

### 앱 다이렉트 발송 (방법 1, 권장)
```
학교: 결과 저장 → "🚀 방법 1" 탭 → "앱 다이렉트 발송" 1클릭
  ↓
mock_edu_receipt/{시도}/ 에 .json 저장 + school_storage/{학교코드}/_outbox/ 발송 기록
  ↓
교육청: 수신함에서 상세 조회 → 자동 read marker 저장
  ↓
학교: 결과 저장 또는 데이터 전송 페이지 재방문 → "✅ 수신 확인 완료" 자동 반영
```

조건: 학교 PC 와 교육청 PC 가 **같은 SafeLoop 인스턴스 또는 공유 NAS / 같은 클라우드 URL** 환경.
